# Day 12 - Time and Space Complexity

## What is Time Complexity?
Time Complexity represents the amount of time taken by an algorithm as a function of input size (n).

Example:
- O(1) → Constant Time
- O(n) → Linear Time
- O(n^2) → Quadratic Time

## What is Space Complexity?
Space Complexity represents the amount of memory (space) taken by an algorithm as a function of input size.

## Key Points Learned:
- How to measure efficiency of algorithms
- Difference between time and space complexity
